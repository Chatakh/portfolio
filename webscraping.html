Movie Trend with Web Scraping

(Capstone Project - OurSecondInnings.org)

Project to scrap web contents off movie review site rottentomatoes.com

Web Scraping is a data product built in Python, where I scrapped the movie contents from the movie review website ‘rottentomatoes’ and stored all the data in a structured way into a csv file format and then read again using python’s library Pandas. I used Python and it’s library BeautifulSoup and Pandas for this project. Web Scraping is very useful for the marketing purpose. Check the YouTube video Movie Trend with Web Scraping

➣ Analysis of data elements of the rottentomatoes.com web page.
➣ Created the data flow diagram of the project.
➣ Script/program to scrap and clean the data elements.
➣ Store scrapped data into a comma delimited .csv file.
➣ Using Pandas, read and display the scrapped structured data from .csv on console.

Tools used: Jupyter, Python with library Pandas and BeautifulSoup.
